## What I build

**Neural Content Intelligence** — [TRIBE v2](https://github.com/rippere/tribe-v2) scores video by predicted cortical attention (fMRI-trained), then *honestly tests itself*: a pre-registered incremental-validity study with negative controls and creator-grouped cross-validation. The science is the spine, not the garnish.

**Decision Intelligence** — AI systems that don't just answer questions, but surface blind spots. Executive Mind Matrix deploys adversarial agent dialectics (Entrepreneur vs. Quant vs. Auditor) so organizations get risk-adjusted recommendations instead of single-model outputs.

**Agentic Infrastructure** — Alfred is a self-hosted butler that runs ambient workflows without prompting. Continuous inbox processing, vault health sweeps, knowledge graph maintenance. You set it up, then stop thinking about it.

**AI-native CRM** — NovaCRM ingests communications from Gmail and Slack, scores leads semantically using pgvector, transcribes calls via Whisper, and surfaces deal health signals before they show up in manual review.
